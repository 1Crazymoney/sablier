## Eslint Config

Eslint config to be shared across Sablier projects and packages.

## Usage

Install the module:

```bash
$ yarn add @sablier/eslint-config
```

And extend it in your `.eslintrc` file:

```js
{
    "extends": ["@sablier/eslint-config"]
}
```
